# Offboarding Checklist (Google Workspace + Slack + Trello)

## Inputs needed
- User email
- Last working date/time (important for immediate access removal)
- Manager confirmation of what to retain/transfer
- Any active projects/ownership (files, boards)

## 1) Immediate access control
- [ ] Suspend Google Workspace account immediately (or at agreed time)
- [ ] Force sign-out / revoke sessions (if available in your plan/policy)
- [ ] Reset password (optional, depending on policy)

## 2) Data preservation and transfer
- [ ] Transfer ownership of critical Google Drive files (to manager/service account)
- [ ] Check Shared Drives for ownership/manager roles
- [ ] Preserve email data if required (Google Vault / retention policy)

## 3) Remove access (Groups-first)
- [ ] Remove user from all Google Groups (or remove from privileged groups first)
- [ ] Ensure no direct permissions remain on files/folders (avoid exceptions)

## 4) Slack
- [ ] Deactivate user
- [ ] Transfer ownership of key channels/apps if needed
- [ ] Export/retain data per company policy (if applicable)

## 5) Trello
- [ ] Remove user from boards
- [ ] Transfer cards/ownership to manager or team lead
- [ ] Archive or reassign outstanding tasks

## 6) Final verification
- [ ] Confirm user cannot access Google services
- [ ] Confirm user removed from key systems (Slack/Trello)
- [ ] Document actions taken and close ticket
