# Google Workspace IT Ops Portfolio

Practical work samples for IT Operations / IT Administration roles focused on Google Workspace, access control, onboarding/offboarding, and basic automation.

## Contents
- 01-access-model-google-workspace/
  - Google Groups strategy for Shared Drives + permissions matrix
- 02-onboarding-offboarding-sop/
  - Onboarding and offboarding checklists (Google Workspace + Slack + Trello)
- 03-automation-example/
  - No-code automation workflow outline (Make/Zapier/n8n style)
- 04-scripts/
  - Small Python script example (CSV -> proposed group membership actions)

## What this demonstrates
- Role-based access control using Groups (auditable, scalable)
- Clean onboarding/offboarding process design
- Automation mindset with basic error handling and logging
- Simple scripting for repeatable admin tasks

## Notes
These samples use realistic examples and naming conventions. In a real environment, group names, permissions, and workflows should align with company policy and compliance requirements.
